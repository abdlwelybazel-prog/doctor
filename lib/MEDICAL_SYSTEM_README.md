# دليل استخدام نظام الاستشارات الطبية المحسّن

## 🏥 نظرة عامة

تطبيق الاستشارات الطبية الشامل مع نظام ذكي لتحليل الحالات الصحية واقتراح الأدوية والأطباء المناسبين.

---

## 🚀 البدء السريع

### 1. **تسجيل دخول الأدمن**
```
البريد الإلكتروني: admin@digl.com
كلمة المرور: Admin@2025Secure
```
يتم إنشاء هذا الحساب تلقائياً عند أول تشغيل للتطبيق.

### 2. **تسجيل حساب مريض جديد**
1. اختر "إنشاء حساب جديد"
2. اختر نوع الحساب "مريض"
3. ملء البيانات الأساسية
4. عند الدخول للصفحة الرئيسية للمرة الأولى، سيُطلب منك ملء الأسئلة الصحية

### 3. **ملء الملف الصحي**
اجب على الأسئلة الـ 6:
- العمر
- الجنس
- هل تعاني من مرض مزمن؟
- اوصف الأعراض الحالية
- متى بدأت الأعراض؟
- درجة الألم (1-10)

---

## 📊 كيفية عمل نظام التحليل

### المرحلة 1: جمع البيانات
يتم جمع معلومات صحية شاملة عن المريض في نموذج منظم وسهل.

### المرحلة 2: التحليل الذكي
```
تحليل البيانات
├─ حساب مستوى الخطورة
├─ تحليل الأعراض
├─ البحث عن أدوية مناسبة
└─ تحديد التخصصات الطبية
```

### المرحلة 3: عرض النتائج
- تقييم مستوى الخطورة بألوان واضحة
- أدوية مقترحة مع جرعات وآثار جانبية
- تخصصات طبية موصى بها
- إجراءات فورية
- تحليل مفصل

---

## 💊 قاعدة بيانات الأدوية

### الأدوية المتوفرة حالياً:

#### 1. الزكام والإنفلونزا
- **بنادول كولد**
  - المادة الفعالة: باراسيتامول + كافيين
  - الجرعة: 500 ملغ كل 4-6 ساعات
  - التحذيرات: لا تتجاوز 3000 ملغ يومياً

- **ديسبرين**
  - المادة الفعالة: ديكستروميثورفان + باراسيتامول
  - الجرعة: ملعقة صغيرة كل 4 ساعات
  - الآثار الجانبية: دوخة، رغبة في النعاس

#### 2. الحساسية
- **كلاريتين**
  - المادة الفعالة: لوراتاديين
  - الجرعة: حبة واحدة يومياً
  
- **أكوافين**
  - المادة الفعالة: سيتريزين
  - الجرعة: حبة واحدة يومياً

#### 3. الجهاز الهضمي
- **رينيول** - للحموضة والحرقة
- **ميوكوستا** - لحماية المعدة

#### 4. الألم والالتهاب
- **ايبوبروفين** - مسكن ومضاد التهاب
- **أسبرين** - مسكن خفيف

#### 5. مشاكل النوم
- **ميلاتونين** - منوم طبيعي

## 👨‍⚕️ التخصصات الطبية المتوفرة

1. **طب عام**
   - للأعراض العامة والفحوصات الروتينية

2. **أنف وأذن وحنجرة**
   - للزكام والسعال والتهاب الحلق

3. **جلدية**
   - للطفح الجلدي والحساسية وحب الشباب

4. **طب الجهاز الهضمي**
   - للمشاكل الهضمية والحموضة

5. **طب الأعصاب**
   - للصداع والدوخة والأعراض العصبية

6. **القلب والأوعية الدموية**
   - لمشاكل القلب والضغط والسمنة

## ⚙️ الإعدادات والتخصيصات

```dart
await AdminSetupService.changeAdminPassword(
  adminEmail: 'admin@digl.com',
  currentPassword: 'Admin@2025Secure',
  newPassword: 'NewPassword123!',
);
```

### إضافة صلاحيات جديدة:
```dart
await AdminSetupService.addPermissionToAdmin(
  adminId: 'admin_id',
  permission: 'manage_new_feature',
);
```

### إلغاء تفعيل حساب أدمن:
```dart
await AdminSetupService.toggleAdminStatus(
  adminId: 'admin_id',
  isActive: false,
);
```

## 🔍 استكشاف الأخطاء

### المشكلة: لا يتم عرض النتائج
**الحل:**
1. تأكد من اتصالك بالإنترنت
2. تأكد من ملء جميع الأسئلة بشكل صحيح
3. حاول إعادة التحميل

### المشكلة: حساب الأدمن لم يتم إنشاؤه
**الحل:**
1. تحقق من تسجيل الأخطاء في وحدة تحكم Android Studio
2. تأكد من وجود إذن الكتابة في Firestore
3. حاول حذف التطبيق وإعادة تثبيته

### المشكلة: توصيات الأدوية غير دقيقة
**تذكير:**
- التحليل استرشادي فقط
- يجب استشارة طبيب متخصص
- البيانات الطبية تحتاج تحديث مستمر

## 🎓 نصائح الاستخدام الأمثل

### للمريض:
1. **ملء البيانات بدقة** - كلما كانت البيانات دقيقة، كانت النتائج أفضل
2. **وصف الأعراض بوضوح** - استخدم مصطلحات طبية صحيحة
3. **اتبع الإجراءات الفورية** - لا تتجاهل التحذيرات
4. **استشر الطبيب دائماً** - لا تعتمد على التطبيق وحده

### للأدمن:
1. **راجع الطلبات بسرعة** - الأطباء ينتظرون الموافقة
2. **احتفظ بسجلات دقيقة** - للمرجعية المستقبلية
3. **حدّث قاعدة البيانات** - أضف أدوية وتخصصات جديدة
4. **راقب الإحصائيات** - تتبع نمو التطبيق
