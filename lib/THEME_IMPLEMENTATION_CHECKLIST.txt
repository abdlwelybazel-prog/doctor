الخطوات لكل ملف:
1. أضف الاستيراد:
   import '../../core/config/medical_theme.dart';
   import '../../core/config/theme_helper.dart';

2. استبدل الألوان الثابتة:
   Colors.white → MedicalTheme.pure
   Colors.blue → MedicalTheme.primaryMedicalBlue
   Colors.red → MedicalTheme.dangerRed
   الخ...

3. استخدم Helper للرسائل:
   ThemeHelper.showErrorSnackBar(context, message);
   ThemeHelper.showSuccessSnackBar(context, message);

4. تحقق من الثيم:
   isDark = ThemeHelper.isDarkMode(context);
